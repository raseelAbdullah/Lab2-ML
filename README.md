# Lab2-ML
Dataset Summary:
The dataset used in this project is the Titanic dataset obtained from Kaggle. 
It contains information about passengers such as age, gender, ticket class, fare, and survival status.

Machine Learning Problem:
This is a supervised classification problem.
The target variable is "Survived", which indicates whether a passenger survived (1) or not (0).

Problem Type:
Classification (Binary Classification)

Objective:
The goal of the model is to predict whether a passenger survived the Titanic disaster based on their personal and travel information.
